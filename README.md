# Компьютерная графика и 3D-моделирование

Репозиторий содержит реализацию классических алгоритмов компьютерной графики на Python. Все алгоритмы реализованы с использованием библиотек matplotlib и numpy для визуализации результатов.

## Структура проекта

- Bresenham - Растровые алгоритмы построения примитивов
- Cyrus-Beck - Алгоритм отсечения отрезков
- Filling - Алгоритмы заливки замкнутых областей
- 2D and 3D - Аффинные преобразования в 2D и 3D пространстве
- Open3D - Работа с трехмерными сетками
- Roberts - Удаление невидимых линий и поверхностей


## Установка зависимостей

pip install matplotlib numpy open3d

## Описание алгоритмов

### 1. Алгоритм Брезенхема

[Брезенхем](https://github.com/Adam-dunno/3d_model/tree/main/Bresenham)

Классический растровый алгоритм для построения отрезков и окружностей на экране. Использует только целочисленную арифметику, что делает его быстрым и эффективным.

Файлы:
- line - построение прямых линий
- circle - построение окружностей

Принцип работы: Алгоритм пошагово определяет, какие пиксели нужно закрасить для отображения линии, минимизируя вычислительные затраты за счет использования накапливаемой ошибки.

### 2. Отсечение по алгоритму Цируса-Бека

[Цирус-Бек](https://github.com/Adam-dunno/3d_model/tree/main/Cyrus-Beck)

Параметрический алгоритм отсечения отрезков произвольным выпуклым многоугольником. Более универсален по сравнению с алгоритмом Коэна-Сазерленда.

Основная идея: Отрезок представляется в параметрической форме, затем вычисляются точки пересечения с границами области отсечения. Определяется видимая часть отрезка на основе нормалей к сторонам многоугольника.

### 3. Алгоритм граничной заливки

[Заливка с методом](https://github.com/Adam-dunno/3d_model/tree/main/Filling)

Заполнение замкнутой области начиная с заданной точки до достижения границы определенного цвета.

Механизм: Рекурсивно или итеративно проверяются соседние пиксели. Если пиксель не является границей и еще не закрашен, он окрашивается в целевой цвет, и процесс повторяется для его соседей.

### 4. Геометрические преобразования

[2d и 3d](https://github.com/Adam-dunno/3d_model/tree/main/2d%20and%203d)

Реализация базовых аффинных преобразований для двумерных и трехмерных объектов:
- Перенос
- Масштабирование
- Поворот
- Отражение

Применение: Преобразования задаются матрицами, что позволяет комбинировать их и применять к множеству точек одновременно.

### 5. Работа с 3D-мешами в Open3D

[Open3D](https://github.com/Adam-dunno/3d_model/tree/main/Open3D)

Загрузка, обработка и визуализация трехмерных сеток. Включает триангуляцию поверхностей и различные методы рендеринга.

Возможности:
- Чтение mesh-моделей различных форматов
- Применение текстур и материалов
- Вычисление нормалей к поверхностям
- Интерактивная визуализация

### 6. Алгоритм Робертса

[Roberts](https://github.com/Adam-dunno/3d_model/tree/main/Roberts)

Метод удаления невидимых линий и поверхностей для корректного отображения трехмерных объектов.

Суть метода: Для каждой грани вычисляется нормаль, затем проверяется ее ориентация относительно наблюдателя. Грани, обращенные от наблюдателя, не отображаются.
